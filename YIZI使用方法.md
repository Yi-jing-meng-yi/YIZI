# YIZI-亦子使用方法  
**新一代安全保护措施**  

## 什么是亦子？  
- 保护您的隐私  
- 实现权限匿名化  
- 解决应用/网页过度申请权限的问题  

## 为什么选择YIZI？  
与其他模拟空间的对比：  

| 模式       | 效果                         | 易用性   |  
|------------|------------------------------|----------|  
| 模拟空间   | 可权限匿名化                 | 较繁琐   |  
| YIZI       | 匿名化+修改权限返回的数据    | 简单     |  

## 使用前提  
支持的安卓版本及模式：  

| 安卓版本       | 模式         | 依赖项       | 是否可用               |  
|----------------|--------------|--------------|------------------------|  
| Android <7.0   | root         | 无           | ❌ 不支持              |  
| Android <7.0   | adb          | 无           | ❌ 不支持              |  
| Android ≥7.0   | root         | 无           |  可用                |  
| Android ≥7.0   | 有线调试     | Shizuku      |  可用                |  
| Android ≥10.0  | adb          | Shizuku      |  可用                |  

## 如何查看安卓版本？  
以真我手机为例：  
1. 进入 **设置**  
2. 选择 **关于手机**  
3. 查看 **Android版本**  

## 如何开启ADB调试？  
以Shizuku工具为例（需支持无线调试）：  
1. 进入 **开发者选项**  
2. 启用 **无线调试**  
3. 打开Shizuku应用  
4. 按照提示返回开发者选项完成配对  
5. 输入配对码  
6. 返回Shizuku，完成激活
